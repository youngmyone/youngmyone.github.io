1. 安装hexo

首先什么是Hexo

Hexo是一个快捷、简洁、高效的博客框架。Hexo使用Markdown解析文章，在几秒内，即可利用靓丽的主题生成静态网页。

详细的可以查看hexo官方文档。

安装

前提：

  • Node.js (版本至少是6.9)
  • Git

保证安装上述程序后，可以使用npm完成Hexo安装

2. 配置GitHub

创建github仓库

上面建站完成后，还只是在本地上能够查看，下面我们将其上传到github上，能够进行互联网访问。

首先得有github账号，然后登录github，新建一个你的用户名.github.io的仓库。比如，你的用户名是test，那么这个仓库名就是test.github.io，其他名称都是无效的。所以访问的地址也就是test.github.io。
